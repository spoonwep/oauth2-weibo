# oauth2-weibo
Weibo OAuth 2.0 support for the PHP League's OAuth 2.0 Client
##Install
You can open a terminal and type in
```shell
composer require spoonwep/oauth2-weibo
```
or require in a composer.json
```json
"require": {
	"spoonwep/oauth2-weibo": "^1.0"
}
```
then run:
```shell
composer update
```
